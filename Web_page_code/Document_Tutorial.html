Mc FUNCTION使用指南

阶段一：文件查找与准备

此教程只适用于国际基岩版、Java 版不适用，请自行上网搜索。

在文件管理器中找到“storage/emulated/0/下载/Document”路径，选中“mc_particle_1.mcfunction”文件与“McFunctionPack.zip”压缩包 。若在此路径未找到，需先确定文件实际位置 。

阶段二：文件解压与整合

  • 1. 对“McFunctionPack.zip”执行解压操作，得到“McFunctionPack”文件夹及相关文件。
  • 2. 剪切（或复制）“mc_particle_1.mcfunction”文件，打开“McFunctionPack”文件夹里的“functions”子文件夹，在“functions”文件夹中粘贴“mc_particle_1.mcfunction”文件，并确认移入成功。

阶段三：文件放置到游戏目录

  • 1. 选中“McFunctionPack”文件夹，进行剪切（或复制）。
  • 2. 依次进入“storage/emulated/0/Android/data/ com.mojang.minecraftpe/files/games/ com.mojang/behavior_packs”路径，将“MyFunctionPack”文件夹粘贴到“behavior_packs”文件夹中，确认移动（或复制）成功。

阶段四：游戏内激活与执行

  • 1. 打开《我的世界》游戏，进入“设置” - “全局资源”界面。
  • 2. 在“行为包”分类中找到“McFunctionPack”，点击“激活”使其生效。
  • 3. 进入游戏世界，在聊天框中输入“/function 替换为实际文件名”（如果文件名是mc_particle_1.mcfunction ，一般去掉后缀使用 ），按下回车键，即可执行该函数对应的游戏功能。原版是支持写到命令方块执行
